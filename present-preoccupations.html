This site's purpose: to teach me first principles of front-end web dev. I've been steeped in back-end/algorithms since I can remember; I got recently got tired of my lack of deep front-end understanding. So I built this and kept it "pure", anti-framework. Just a Python webserver with minimal usage of Express and Jquery. I've learned a ton, but admittedly, the site is a bit ugly. Now that an MVP is up, my current tasks are improved design, moving to REACT (?), more detailed project pages (ft. live demos), and more.
